# 大学生月度消费分析

> 该份学生消费数据是通过自制模拟数据而来

## 一、选题介绍

大学生消费行为是反映学生生活状态和经济观念的重要指标。通过分析大学生的月度消费数据，可以了解不同年级学生的消费水平差异、消费结构特点，为高校学生管理、家庭教育和学生个人理财规划提供参考依据。

## 二、数据来源

### 2.1 数据文件

数据来自 `student.csv` 原始数据文件，包含学生基本信息和月度消费记录。

### 2.2 数据结构

| 字段 | 说明 |
|------|------|
| 年级 | 大一、大二、大三、大四 |
| 学号 | S001-S060 |
| 性别 | 男、女 |
| 餐饮支出 | 每月餐饮支出金额 |
| 娱乐支出 | 每月娱乐支出金额 |
| 其它支出 | 每月其它支出金额 |
| 月总支出 | 每月总支出金额 |

### 2.3 样本统计

- **总样本量**：61份
- **有效样本量**：54份
- **无效样本原因**：
  - 缺失值：缺少对应样本的数据
  - 重复值：完全一致的样本
  - 异常值：数据不在合理范围内
  - 数据校验不一致：数据校验结果与预期不符

## 三、分析过程

### 3.1 年级消费分析（柱状图）

通过柱状图分析发现：

- 年级各总月总支出随年级递增
- 大一消费为 20000~25000 区间
- 大二和大三消费为 25000~30000 区间
- 大四消费为 30000 以上

**结论**：
- 从大一到大四，消费水平呈现 **逐级递增** 趋势
- 这符合大学生随着年级增长，社交活动、学习资料、实习交通等消费需求增加的实际情况

### 3.2 消费结构分析（饼图）

通过饼图分析发现：

餐饮支出远超娱乐和其它支出，娱乐其次，其它最小。

| 消费类别 | 占比 | 分析 |
|----------|------|------|
| 餐饮支出 | 64.2% | 符合大学生日常生活消费特点，饮食是最基本的需求 |
| 娱乐支出 | 21.5% | 表明大学生在满足基本生活需求后，有相当比例的消费用于休闲娱乐 |
| 其它支出 | 14.2% | 其它或许则可能如购买图书学习、日常用品等 |

## 四、结论总结

### 4.1 年级消费差异显著

- 消费水平随年级增长呈上升趋势
- 大四学生消费最高，大一学生消费最低
- 年级越高，消费需求越多元化

### 4.2 消费结构特点

- 餐饮支出是最大消费项目（45.3%）
- 娱乐消费占比较高（32.7%）
- 学习消费占比相对较低（22.0%）
